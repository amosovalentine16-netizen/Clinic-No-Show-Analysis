Clinic Appointment No-Show Analysis
 Project Overview

This project analyzes clinic appointment data to identify patterns in patient no-shows. The objective is to uncover trends across days of the week and departments to support operational decision-making.

Tools Used

Python

Pandas

Matplotlib

Jupyter Notebook

 Key Findings

Overall no-show rate: 20%

Highest no-show days: Friday and Tuesday (29.4%)

Highest no-show department: Dental (35.7%)

Lowest no-show department: General (7.14%)

 Recommendations

Implement targeted reminders for high-risk departments.

Adjust scheduling strategies for high-risk days.

Study mid-week attendance stability for operational insights.
